\begin{DoxyEnumerate}
\item \href{https://github.com/Fazanwolf/R-Type/fork}{\texttt{ Fork}} and clone the repository
\item Create a new branch\+: {\ttfamily git checkout -\/b feature/name-\/feature}
\item Build the project {\ttfamily cmake -\/B build -\/S . \&\& cmake -\/-\/build build -\/-\/config Release} and run the tests {\ttfamily cmake -\/-\/build build -\/-\/config Release -\/-\/target test}
\item Make your change, add tests, and make sure the tests still pass
\item Push to your fork and \href{https://github.com/Fazanwolf/R-Type/pulls}{\texttt{ submit a pull request}}
\item Pat yourself on the back and wait for your pull request to be reviewed and merged.
\end{DoxyEnumerate}

Here are a few things you can do that will increase the likelihood of your pull request being accepted\+:


\begin{DoxyItemize}
\item Write tests.
\item Keep your change as focused as possible. If there are multiple changes you would like to make that are not dependent upon each other, consider submitting them as separate pull requests.
\item Write a \href{https://gist.github.com/luismts/495d982e8c5b1a0ced4a57cf3d93cf60}{\texttt{ good commit message}}.
